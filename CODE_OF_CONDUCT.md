# Code of Conduct

This project uses a simple standard:

- Be rigorous.
- Be direct.
- Be respectful.
- Critique ideas and evidence, not people.
- Do not use personality labels to demean, exclude, diagnose, or control others.

Unacceptable behavior includes harassment, identity-based attacks, threats, doxxing, and using this project to justify high-stakes selection or discrimination.

Report concerns through GitHub issues or by contacting the repository owner.
