# Prompt Recipes

Copy one of these prompts into Codex after installing the skill. They are written to produce a high-signal first session quickly.

## Live Typing: No Quick Label

```text
Use $mbti-typing. I want a rigorous MBTI typing interview.

Do not give me a quick label. Start with 3-6 candidate types, ask 4-6 targeted questions per round, keep a serious runner-up, and update the evidence board after each round.

My current claim is: [PASTE YOUR CLAIM OR TEST RESULTS HERE]
My biggest contradiction is: [PASTE CONTRADICTION HERE]
```

## Adjacent-Type Duel

```text
Use $mbti-typing to distinguish [TYPE A] vs [TYPE B].

Do not ask generic introversion/extroversion questions. Build a pair duel with two independent discriminators, one stress/recovery probe, and one counterexample question. End with what evidence would make you switch sides.
```

## Transcript Audit

```text
Use $mbti-typing to audit these exported conversations.

Extract the candidate set, recurring evidence, contradictions, user corrections, overclaims, and the best current formulation. Preserve runner-up types. Separate normal state, stress state, relationship state, and public-performance state.

Files or excerpts:
[PASTE OR ATTACH CONVERSATION EXPORTS]
```

## Report Review

```text
Use $mbti-typing to review this MBTI report.

Prioritize bugs in reasoning: missing runner-up, unsupported certainty, weak evidence, framework mixing, missing falsifiers, and single-anecdote overfitting. Give findings first, then a cleaner revised formulation.

Report:
[PASTE REPORT]
```

## Chinese High-Retention Interview

```text
Use $mbti-typing。请用中文做高穿透力判型，但不要奉承、不要玄学、不要假确定。

每轮只打一个关键分叉：先列候选集，再问 4-6 个问题；回答后更新候选榜、最强证据、最大矛盾、下一轮目标。最终必须写 runner-up 和什么证据会让我改判。

我的自述：
[粘贴你的自述]
```

## Long Session With State

```text
Use $mbti-typing for a long MBTI typing session.

Create or update a structured session state using the schema in references/session-state.md. After each round, preserve candidate_set, evidence, contradictions, user_corrections, current_board, next_round_target, and falsifiers.

Target question:
[PASTE TARGET QUESTION]
```
